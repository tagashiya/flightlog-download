# FlightLog - 配布ファイル

ドローン飛行記録管理アプリ「FlightLog」の初期設定に必要なファイルです。

---

## ダウンロード方法

1. このページ右上の緑色の「**Code**」ボタンをクリックする
2. 「**Download ZIP**」をクリックする
3. ダウンロードされたZIPファイルを解凍する
4. 中に入っている以下の2ファイルを使用する
   - `FlightLog_AppsScript.js`（Google Apps Scriptに貼り付けるコード）
   - `利用者向け手順書.txt`（初期設定の手順書）

---

## 初期設定について

詳しい設定手順は同梱の「利用者向け手順書.txt」をご確認ください。

初期設定には以下が必要です。

- Googleアカウント
- 所要時間：約15分
